Solution

题目要求

∑x=1a∑y=1b[gcd(a,b)=d]\sum_{x=1}^a\sum_{y=1}^b[gcd(a,b)=d] x=1∑a​y=1∑b​[gcd(a,b)=d]

设上式为f(d)f(d)f(d)，构造g(n)=∑n∣df(d)g(n)=\sum_{n|d}f(d)g(n)=∑n∣d​f(d)，于是有

g(n)=∑x=1a∑y=1b[n∣gcd(a,b)]g(n)=\sum_{x=1}^a\sum_{y=1}^b[n|gcd(a,b)] g(n)=x=1∑a​y=1∑b​[n∣gcd(a,b)]

易得

g(d)=⌊ad⌋⌊bd⌋g(d)=\lfloor\frac{a}{d}\rfloor\lfloor\frac{b}{d}\rfloor g(d)=⌊da​⌋⌊db​⌋

代入反演一下

f(n)=∑n∣dμ(dn)⌊ad⌋⌊bd⌋f(n)=\sum_{n|d}\mu(\frac{d}{n})\lfloor\frac{a}{d}\rfloor\lfloor\frac{b}{d}\rfloor f(n)=n∣d∑​μ(nd​)⌊da​⌋⌊db​⌋

令t=dnt=\frac{d}{n}t=nd​，代入得

f(n)=∑t=1min(a,b)nμ(t)⌊ant⌋⌊bnt⌋f(n)=\sum_{t=1}^\frac{min(a,b)}{n}\mu(t)\lfloor\frac{a}{nt}\rfloor\lfloor\frac{b}{nt}\rfloor f(n)=t=1∑nmin(a,b)​​μ(t)⌊nta​⌋⌊ntb​⌋

预处理μ\muμ的前缀和，整除分块即可
