「Luogu2522」[HAOI2011]Problem b

problem

Solution

题目要求

∑x=ab∑y=cd[gcd(x,y)=k]\sum_{x=a}^b\sum_{y=c}^d[gcd(x,y)=k] x=a∑b​y=c∑d​[gcd(x,y)=k]

设上式为Ans(a,b,c,d,k)Ans(a,b,c,d,k)Ans(a,b,c,d,k)

不妨来想一想a=1,c=1a=1,c=1a=1,c=1的时候怎么做

其实就跟「Luogu3455」[POI2007]ZAP-Queries一模一样了

略过一大堆式子，当a=c=1a=c=1a=c=1时

Ans(1,b,1,d,k)=∑t=1min(b,d)kμ(t)⌊bkt⌋⌊dkt⌋Ans(1,b,1,d,k)=\sum_{t=1}^{\frac{min(b,d)}{k}}\mu(t)\lfloor\frac{b}{kt}\rfloor\lfloor\frac{d}{kt}\rfloor Ans(1,b,1,d,k)=t=1∑kmin(b,d)​​μ(t)⌊ktb​⌋⌊ktd​⌋

容斥一下可得

Ans(a,b,c,d,k)=Ans(1,b,1,d,k)−Ans(1,a−1,1,d,k)−Ans(1,b,1,c−1,k)+Ans(1,a−1,1,c−1,k)Ans(a,b,c,d,k)=Ans(1,b,1,d,k)-Ans(1,a-1,1,d,k)\\-Ans(1,b,1,c-1,k)+Ans(1,a-1,1,c-1,k) Ans(a,b,c,d,k)=Ans(1,b,1,d,k)−Ans(1,a−1,1,d,k)−Ans(1,b,1,c−1,k)+Ans(1,a−1,1,c−1,k)

愉快地调用就可以了
