BSGS

BSGS可以用于解决一类离散对数问题，一般形如

Ax≡B(modC)A^x\equiv B\pmod C Ax≡B(modC)

其中CCC为质数

它是怎么做的？

我们令m=⌈C⌉m=\lceil\sqrt{C}\rceilm=⌈C​⌉，那么x=im+j,i∈[0,m−1],j∈[0,m−1]x=im+j,i\in[0,m-1],j\in[0,m-1]x=im+j,i∈[0,m−1],j∈[0,m−1]

于是原方程转化为

Aim×Aj≡B(modC)A^{im}\times A^j\equiv B\pmod C Aim×Aj≡B(modC)

即

Aj≡B×A−im(modC)A^j\equiv B\times A^{-im}\pmod C Aj≡B×A−im(modC)

我们知道gcd(A,C)=1gcd(A,C)=1gcd(A,C)=1，所以AAA在模CCC意义下存在逆元，AkA^kAk也是

那么我们将所有的AjA^jAj存进哈希表里(Baby Step)(Baby\space Step)(Baby Step)，然后枚举i(Giant Step)i(Giant\space Step)i(Giant Step)，如果找到一个iii满足上式，那么我们就找到了一个解x=i×m+jx=i\times m+jx=i×m+j

如果iii在[0,m−1][0,m-1][0,m−1]内没有解，那么原方程无解
因为根据欧拉定理，Aφ(C)=AC−1≡1(modC),AC≡A(modC)A^{\varphi(C)}=A^{C-1}\equiv1\pmod C,A^C\equiv A\pmod CAφ(C)=AC−1≡1(modC),AC≡A(modC)，出现循环节
所以原方程如果有解，最小的解一定在[0,C−1][0,C-1][0,C−1]范围内，否则无解

时间复杂度O(C)O(\sqrt{C})O(C​)

但是我太菜了，不会写哈希表，只会用mapmapmap，时间复杂度到了O(Clog⁡C)O(\sqrt C\log\sqrt C)O(C​logC​)，才能勉强维持一下生活这样子

BSGS是一种典型的空间换时间的思想
