「CF338D」GCD Table

problem

题意

有一张n×mn\times mn×m的表，其中第iii行第jjj列的元素是gcd(i,j)gcd(i,j)gcd(i,j)。

给定一个长度为kkk的序列aaa，询问在表中是否存在一对(x,y)(x,y)(x,y)使得∀i∈[1,k],ai=gcd(x,y+i−1)\forall i\in[1,k],a_i=gcd(x,y+i-1)∀i∈[1,k],ai​=gcd(x,y+i−1)

在确定了xxx的值之后，还需要满足ai∣y+i−1a_i∣y+i−1ai​∣y+i−1，即满足下列同余方程：
