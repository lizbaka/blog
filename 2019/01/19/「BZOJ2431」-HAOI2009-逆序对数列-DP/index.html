「BZOJ2431」 [HAOI2009]逆序对数列 DP

发表于2019-01-19|更新于2021-12-03|算法竞赛
|字数总计:601|阅读时长:2分钟|阅读量:

problem

Description

对于一个数列ai{ai}ai，如果有i<ji<ji<j且ai>aja_i>a_jai​>aj​，那么我们称aia_iai​与aja_jaj​为一对逆序对数。若对于任意一个由1...n1...n1...n自然数组成的数列，可以很容易求出有多少个逆序对数。那么逆序对数为k的这样自然数数列到底有多少个？

Input

第一行为两个整数nnn，kkk。

Output

写入一个整数，表示符合条件的数列个数，由于这个数可能很大，你只需输出该数对100001000010000求余数后的结果。

Sample Input

Sample Output

Hint

n<=1000,k<=1000n<=1000,k<=1000n<=1000,k<=1000

Solution

考虑一个1...i1...i1...i的排列，当要插入i+1i+1i+1时，由于它是最大的，所以无论在何处插入，其后方的所有数都会与它产生一对新的逆序对

于是有方程dp[i][j]dp[i][j]dp[i][j]表示1...i1...i1...i的排列中，有jjj个逆序对的方案数

转移方程为：

dp[i][j]=∑k=0min(i−1,j)dp[i−1][j−k]dp[i][j]=\sum_{k=0}^{min(i-1,j)}dp[i-1][j-k] dp[i][j]=k=0∑min(i−1,j)​dp[i−1][j−k]

边界为dp[1][0]=1dp[1][0]=1dp[1][0]=1

复杂度是O(nk2)O(nk^2)O(nk2)，注意到可以利用前缀和优化，于是使得复杂度进一步下降为O(nk)O(nk)O(nk)，复杂度正确
