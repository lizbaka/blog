「Luogu2221」[HAOI2012]高速公路

problem

题目描述

Y901Y901Y901高速公路是一条重要的交通纽带，政府部门建设初期的投入以及使用期间的养护费用都不低，因此政府在这条高速公路上设立了许多收费站。

Y901Y901Y901高速公路是一条由N−1N-1N−1段路以及NNN个收费站组成的东西向的链，我们按照由西向东的顺序将收费站依次编号为111~NNN，从收费站iii行驶到i+1i+1i+1(或从i+1i+1i+1行驶到iii)需要收取ViV_iVi​的费用。高速路刚建成时所有的路段都是免费的。

政府部门根据实际情况，会不定期地对连续路段的收费标准进行调整，根据政策涨价或降价。

无聊的小AAA同学总喜欢研究一些稀奇古怪的问题，他开车在这条高速路上行驶时想到了这样一个问题:对于给定的l,r(l<r)l,r(l<r)l,r(l<r),在第lll个到第rrr个收费站里等概率随机取出两个不同的收费站aaa和bbb，那么从aaa行驶到bbb将期望花费多少费用呢?

输入输出格式

输入格式：

第一行222个正整数N,MN,MN,M，表示有NNN个收费站，MMM次调整或询问

接下来M行，每行将出现以下两种形式中的一种

C l r v
表示将第lll个收费站到第rrr个收费站之间的所有道路的通行费全部增加vvv

Q l r

所有CCC与QQQ操作中保证1≤l<r≤N1\le l < r\le N1≤l<r≤N

输出格式：

对于每次询问操作回答一行，输出一个既约分数

若答案为整数aaa，输出a/1

Solution

讲真

这个题除了最终计算答案稍微要用一下期望

跟期望基本没有关系（绝望）

首先，对于每次询问，在区间[l,r][l,r][l,r]中，任选两个点显然有Cr−l+12C_{r-l+1}^{2}Cr−l+12​种选法，每种选法的概率都是1Cr−l+12\frac{1}{C_{r-l+1}^{2}}Cr−l+12​1​

令所有选法的长度之和为sumsumsum，那么答案即为

sumCr−l+12\frac{sum}{C_{r-l+1}^{2}} Cr−l+12​sum​

好了上面就是跟期望有关的部分，现在来看sumsumsum如何维护

令viv_ivi​表示收费站i,i+1i,i+1i,i+1之间的的收费

画图可知，对于每次询问l,rl,rl,r，考虑每一段的贡献，有sum=∑i=1r−li∗(r−l−i+1)∗vl+i−1sum=\sum_{i=1}^{r-l}i*(r-l-i+1)*v_{l+i-1}sum=∑i=1r−l​i∗(r−l−i+1)∗vl+i−1​

以l=2,r=6l=2,r=6l=2,r=6为例，那么sumsumsum即为1∗4∗v2+2∗3∗v3+3∗2∗v4+4∗1∗v51*4*v_2+2*3*v_3+3*2*v_4+4*1*v_51∗4∗v2​+2∗3∗v3​+3∗2∗v4​+4∗1∗v5​

显然这玩意很难维护，我们来颓柿子

∑i=1r−li∗(r−l−i+1)∗vl+i−1\sum_{i=1}^{r-l}i*(r-l-i+1)*v_{l+i-1} i=1∑r−l​i∗(r−l−i+1)∗vl+i−1​

=(r−l+1)∑i=1r−li∗vl+i−1−∑i=1r−li2∗vl+i−1=(r-l+1)\sum_{i=1}^{r-l}i*v_{l+i-1}-\sum_{i=1}^{r-l}i^2*v_{l+i-1} =(r−l+1)i=1∑r−l​i∗vl+i−1​−i=1∑r−l​i2∗vl+i−1​

然而柿子中的各项依然比较难维护，因为每次lll都是不同的，我们来继续变形

∑i=1r−li∗vl+i−1=∑i=1r−l(l+i−1)∗vl+i−1−(l−1)∑i=1r−lvl+i−1\sum_{i=1}^{r-l}i*v_{l+i-1}=\sum_{i=1}^{r-l}(l+i-1)*v_{l+i-1}-(l-1)\sum_{i=1}^{r-l}v_{l+i-1} i=1∑r−l​i∗vl+i−1​=i=1∑r−l​(l+i−1)∗vl+i−1​−(l−1)i=1∑r−l​vl+i−1​

这样我们只需要维护i∗vii*v_ii∗vi​和viv_ivi​的区间和即可求出∑i=1r−li∗vl+i−1\sum_{i=1}^{r-l}i*v_{l+i-1}∑i=1r−l​i∗vl+i−1​

这不禁令我们想到我们是否可以通过维护i2∗vii^2*v_ii2∗vi​的区间和来处理剩下的一项

首先经过变形可得(l+i−1)2=(l−1)2+(2l−2)i+i2(l+i-1)^2=(l-1)^2+(2l-2)i+i^2(l+i−1)2=(l−1)2+(2l−2)i+i2

于是有

∑i=1r−li2∗vl+i−1=∑i=1r−l(l+i−1)2∗vl+i−1−(l−1)2∑i=1r−lvl+i−1−(2l−2)∑i=1r−li∗vl+i−1\sum_{i=1}^{r-l}i^2*v_{l+i-1}=\sum_{i=1}^{r-l}(l+i-1)^2*v_{l+i-1}-(l-1)^2\sum_{i=1}^{r-l}v_{l+i-1}-(2l-2)\sum_{i=1}^{r-l}i*v_{l+i-1} i=1∑r−l​i2∗vl+i−1​=i=1∑r−l​(l+i−1)2∗vl+i−1​−(l−1)2i=1∑r−l​vl+i−1​−(2l−2)i=1∑r−l​i∗vl+i−1​

这样我们就又可以通过维护i2∗vii^2*v_ii2∗vi​的区间和求出∑i=1r−li2∗vl+i−1\sum_{i=1}^{r-l}i^2*v_{l+i-1}∑i=1r−l​i2∗vl+i−1​推死我了我去

区间修改及区间求和显然可以通过线段树来维护
