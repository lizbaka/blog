多项式求逆

多项式求逆指的是给定一个多项式F(x)F(x)F(x)，求出一个多项式G(x)G(x)G(x)满足

F(x)∗G(x)≡1(modxn)F(x)*G(x)\equiv1\pmod {x^n} F(x)∗G(x)≡1(modxn)

它是怎么做的？

我们称一个多项式的“度”为其最高次项系数+1+1+1

首先，我们知道当n=1n=1n=1的时候，显然G(x)G(x)G(x)即为F(x)F(x)F(x)的常数项之逆元

我们将原式写成模x⌈n2⌉x^{\lceil\frac n 2\rceil}x⌈2n​⌉意义下的形式：

F(x)∗G(x)≡1(modx⌈n2⌉)F(x)*G(x)\equiv1\pmod {x^{\lceil\frac n 2\rceil}} F(x)∗G(x)≡1(modx⌈2n​⌉)

假设我们已经求出B(x)B(x)B(x)满足

F(x)∗B(x)≡1(modx⌈n2⌉)F(x)*B(x)\equiv1\pmod {x^{\lceil\frac n 2\rceil}} F(x)∗B(x)≡1(modx⌈2n​⌉)

将两个式子相减

G(x)−B(x)≡0(modx⌈n2⌉)G(x)-B(x)\equiv0\pmod{x^{\lceil\frac n 2\rceil}} G(x)−B(x)≡0(modx⌈2n​⌉)

平方一下

G2(x)−2G(x)B(x)+B2(x)≡0(modxn)G^2(x)-2G(x)B(x)+B^2(x)\equiv0\pmod{x^n} G2(x)−2G(x)B(x)+B2(x)≡0(modxn)

两边乘上F(x)F(x)F(x)

G(x)−2B(x)+F(x)B2(x)≡0(modxn)G(x)-2B(x)+F(x)B^2(x)\equiv0\pmod{x^n} G(x)−2B(x)+F(x)B2(x)≡0(modxn)

（这里由于F(x)∗G(x)≡1(modxn)F(x)*G(x)\equiv1\pmod{x^n}F(x)∗G(x)≡1(modxn)，消去了一些部分）

移项整理得

G(x)≡(2−F(x)B(x))B(x)(modxn)G(x)\equiv(2-F(x)B(x))B(x)\pmod{x^n} G(x)≡(2−F(x)B(x))B(x)(modxn)

多项式乘法可以用FFT/NTT加速
